# 🛒 GADGET HUB

GADGET HUB is a modern and responsive E-Commerce website designed for purchasing electronic gadgets and accessories online. The project provides an attractive user interface, product browsing, shopping cart functionality, and local storage-based data management.

🌐 Live Demo: https://ayush555-50.github.io/GADGET-HUB/

---

## 📖 Project Overview

GADGET HUB is developed to provide users with a simple and interactive online shopping experience. The website allows customers to browse gadgets, view product details, add products to the cart, and manage their shopping activities efficiently.

The project is built using front-end web technologies and utilizes Local Storage for storing user and cart data without requiring a backend database.

## 🚀 Installation

1. Clone the repository

```bash
git clone https://github.com/ayush555-50/GADGET-HUB.git
```

2. Open the project folder

```bash
cd GADGET-HUB
```

3. Run the project

Simply open `index.html` in your browser.

## 💻 Usage

1. Visit the homepage.
2. Browse available gadgets.
3. Search products using the search bar.
